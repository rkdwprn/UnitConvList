# Release notes — Play Store (영문 / 한글)

Play Console에서는 **태그를 문구 안에 넣지 않습니다.**  
출시 노트 화면에서 **언어를 선택**(예: English (United States), 한국어)한 뒤, 그 언어용 문구만 입력합니다.

**넣는 방법**
1. Play Console → **출시** → **프로덕션**(또는 테스트 트랙) → **출시 노트**
2. **언어 추가** 또는 기존 언어 선택
3. **English (United States)** 선택 → 아래 "영문" 문구 붙여넣기
4. **한국어** 선택 → 아래 "한글" 문구 붙여넣기

`<en-US>` 같은 태그는 문구에 적지 않아도 됩니다. 언어는 화면에서 선택하는 것으로 구분됩니다.
