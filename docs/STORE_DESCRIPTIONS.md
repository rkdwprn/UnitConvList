## Play Console 입력 위치

1. **Play Console** → **성장** → **스토어 등록정보** → **기본 스토어 등록정보**
2. **간단한 설명**: 위 "간단한 설명" 문구를 **한국어** / **English** 각각 해당 언어 선택 후 입력
3. **전체 설명**: 위 "자세한 설명" 문구를 **한국어** / **English** 각각 해당 언어 선택 후 입력

문자 수 초과 시 일부 문장을 줄여서 사용하시면 됩니다.
